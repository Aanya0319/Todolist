This app helps users manage daily tasks with an easy-to-use interface.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ⏳ View pending tasks
- 📋 Filter tasks (All / Completed / Pending)
- ❌ Delete tasks
- 🎨 Clean and minimal UI
- 📱 Responsive design
